ε-N Method Practice Problems

Introduction

These practice problems are designed to help you master the ε-N definition of convergence for sequences. For each problem:

  • Determine the limit L (if it exists)
  • Prove convergence using the ε-N definition
  • Show |an - L| < ε implies n > N for some N

Basic Problems

  1. Show that the sequence an = 5/n converges using the ε-N method.
  2. Prove that an = 2 + 3/n² converges to 2 using the ε-N definition.
  3. Use the ε-N method to show that an = (n+1)/n converges.
  4. Prove that an = (4n² - n + 7)/(2n² + 3) converges.

Intermediate Problems

  1. Show that an = (3n³ + 2n² - n + 4)/(n³ + 5n) converges using ε-N.
  2. Prove convergence of an = 1 + 2/n + 1/n³ to its limit.
  3. Use ε-N to show that an = sin(n)/n converges to 0. (Hint: Use the fact that |sin n| ≤ 1)
  4. Prove that an = (n² + 3n - 1)/(2n² + n + 4) converges to its limit.

Challenging Problems

  1. Show that an = (√(n² + n) - n)/n converges using ε-N. (Hint: Rationalize the numerator first)
  2. Prove that an = n²e-n converges to 0.
  3. Use ε-N to show convergence of an = ln(n+1)/n to 0. (Hint: Use ln(n+1) < nα for small α > 0)
  4. Prove that the sequence an = n!/nn converges to 0 using ε-N.

Bonus Problems

  1. (Divergence) Show that an = n does NOT converge using the ε-N definition. (Find an ε > 0 such that no N works.)
  2. (Bounded terms) Prove that an = (-1)n/n converges to 0 using ε-N.
  3. Show that an = (n² + (-1)nn)/(n² + 1) converges to 1.

Tips for Solving ε-N Problems

  • Simplify first: Always simplify |an - L| algebraically before working with ε.
  • Common techniques:
    • For rational functions: Divide numerator and denominator by highest power of n
    • For √(n² + n) - n: Rationalize using conjugates
    • For factorials/exponentials: Use known inequalities
  • Finding N: Solve |an - L| < ε for n, then take N larger than that value.

Practice these problems systematically. The ε-N method builds intuition for rigorous convergence proofs!
